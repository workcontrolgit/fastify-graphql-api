# How to build a blazing fast GraphQL API with Node.js, MongoDB and Fastify!

> A Node.js project

## Build Setup

```bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start
```

## Prerequisites

- Nodejs
- MongoDB
